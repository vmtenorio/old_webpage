Home Access Network Intern

Telefonica | March 2018 - September 2018

From March until September in 2018, I participated in the Talentum internship in Telefonica, working in the Home Access Network department. The job involved two different fields testing the latest software and hardware versions of the Home Gateway Units (HGU) and network amplifiers (Baseports) that Telefonica delivers to its clients, and testing new standards in the fields of xDSL and optical fiber. Top results in the market were achieved, with traffic of more than 8.5 Gbps in a single device that was prepared to use one of these optical fiber standards (NGPON-2).

Aerospace Engineering Researcher

Linköping University | June 2017 - September 2017

During the summer of 2017, I went to Linköping, in Sweden, to participate in an Internship in the Linköping University with the Erasmus+ program. In this internship, I participated in a project in which we extracted information from a drone with the Pixhawk controller. To gather, process and store the information, the Pixhawk controller was attached to a Raspberry Pi. In the Pixhawk, the Ardupilot code library was loaded. My job involved modifying the configuration of the Pixhawk and the underlying code to gather the required information and storing it in the Raspberry Pi. The modified code is available in this link.
